# Unused React Component Detector

Find and safely delete unused React components from your codebase.

## Features
- Scan entire project for unused components
- Build dependency graph
- Safety checks before deletion
- One-click safe cleanup

## Installation
1. Install from VS Code Marketplace
2. Open command palette (Cmd+Shift+P)
3. Run "Find Unused Components"

## Usage
Click the status bar button or use command palette.

Built with TypeScript, VS Code Extension API.
