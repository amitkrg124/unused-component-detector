# Change Log

All notable changes to the "Unused React Component Detector" extension will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of Unused React Component Detector
- Component scanning functionality
- Dependency analysis with import tracking
- Safety checks before component deletion
- Webview UI for displaying unused components
- Status bar integration
- Support for TypeScript and JavaScript React components
- Detection of ES6 and CommonJS imports
- Support for dynamic imports
- Test file detection
- Index file export checking
- Configuration file reference detection
- Progress indicators during scanning
- File size and modification date display
- Safety badges and warnings
- One-click component deletion with confirmation

### Features
- Scans entire workspace for React components
- Identifies unused components (zero imports)
- Checks component safety before deletion
- Displays detailed warnings and recommendations
- Integrates with VS Code theme colors
- Real-time panel updates
